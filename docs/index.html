Menu Bar Access

No more digging through System Settings. Toggle your scroll direction with a single click from the comfort of your menu bar.

Visual Indicators

Dynamic icons show your current mode at a glance. Easily tell if you're in Natural or Standard mode.

Native & Fast

Built with Go and native macOS APIs. Lightweight, memory-efficient, and respects system preferences.

Built for the
Hybrid Setup.

Most MacBook users switch between a trackpad and a high-performance mouse daily. macOS treats the scroll setting as global, forcing you to manually dive into System Settings every time you change devices.

Auto-detects current system state on startup.

Zero-configuration background service.

Quick Install

Clone and build Flip in seconds.

Terminal 
# Clone the repository
git clone https://github.com/hsbacot/flip
cd flip

# Build the application
./build.sh

# Move to Applications
mv Flip.app /Applications

Requires Go and a Mac running macOS. See the Full README for details.
